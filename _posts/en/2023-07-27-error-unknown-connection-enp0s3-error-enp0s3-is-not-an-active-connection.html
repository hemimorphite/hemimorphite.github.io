If you encounter the "unknown connection enp0s3" error or "enp0s3 is not an active connection" error in nmcli command, there is a easy solution to fix the issue.

First, delete the connection:

sudo nmcli connection delete enp0s3

Then, re-create the connection:

sudo nmcli connection add type ethernet con-name enp0s3
